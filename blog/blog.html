JavaScript Questions & Answers

1. What is a Closure in JavaScript?

A closure is a function that retains access to its lexical scope, even when the function is executed outside that scope. It allows for data encapsulation and the ability to create private variables.

2. What is the purpose of the this keyword?

The this keyword refers to the object it belongs to. It provides a way to access object properties and methods from within the object itself, or in functions to refer to the calling object.

3. Explain async and await in JavaScript.

The async keyword is used to define an asynchronous function that returns a promise. The await keyword is used inside an async function to wait for the promise's resolution. It makes asynchronous code look and behave like synchronous code.

4. What is a Promise?

A Promise is an object representing the eventual completion or failure of an asynchronous operation. It can be in one of three states: pending, fulfilled, or rejected.

5. What are JavaScript Data Types?

JavaScript data types include primitive types like string, number, boolean, null, undefined, symbol, and bigint, as well as complex types like object and function.

6. What is the difference between == and ===?

The == operator compares two values for equality after converting them to a common type (type coercion). The === operator, also known as the strict equality operator, compares both value and type without performing type coercion.

7. Explain the event loop in JavaScript.

The event loop is a mechanism that allows JavaScript to perform non-blocking operations by handling asynchronous callbacks. It continuously checks the call stack and the task queue, executing queued tasks when the stack is empty.

8. What is the difference between var, let, and const?

var declares a variable with function scope and allows redeclaration. let declares a block-scoped variable and does not allow redeclaration. const declares a block-scoped constant that cannot be reassigned.

9. What are Arrow Functions?

Arrow functions provide a shorter syntax for writing function expressions and do not have their own this, arguments, super, or new.target bindings. They are best suited for non-method functions.

10. What is the purpose of the JSON.stringify() method?

The JSON.stringify() method converts a JavaScript object or value to a JSON string. It is useful for serializing data to be sent over a network or stored in a text format.
